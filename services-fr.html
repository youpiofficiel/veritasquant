Deux modes, trois niveaux de profondeur

Nous proposons deux modes d'engagement. L'audit de performance opère uniquement sur votre historique de trades. L'audit complet ajoute une revue de votre logique de signal et de votre code source, vérifiant l'intégrité de l'implémentation en plus de la validité statistique.

Mode A
Audit de performance
Vous fournissez un historique de trades : horodatages d'entrée/sortie, instrument, direction et prix. Nous validons les propriétés statistiques de vos résultats à travers notre cadre propriétaire. Aucun code requis.
Requis → historique de trades (CSV / export API)
Mode B
Audit complet
Vous fournissez votre historique de trades et le code source du signal (ou un pseudocode détaillé). En plus de la validation statistique, nous auditons l'implémentation pour détecter le biais d'anticipation, les erreurs de construction du signal et les problèmes de traitement des données qui gonflent silencieusement les performances de backtest.
Requis → historique de trades + code source ou pseudocode

Quatre niveaux de profondeur

Mode A
Diagnostic
Filtre statistique · 5–7 jours ouvrés
Validation statistique fondamentale appliquée à votre historique de trades. Tests de significativité, survie aux coûts, suffisance d'échantillon, persistance hors échantillon et confusion directionnelle. Livraison d'un rapport structuré avec des verdicts binaires par couche de test et un commentaire interprétatif.
Test de significativité · Stress coûts · Suffisance d'échantillon · Walk-forward · Détection de confusion · Rapport structuré

Mode A
Validation complète
Cadre complet · 10–15 jours ouvrés
Le cadre de validation propriétaire complet. Inclut un débrief avec l'équipe de recherche.
Toutes les couches diagnostic · Microstructure de marché · Analyse de régime · Diagnostics de décroissance · Rapport DD · Débrief recherche

Mode B
Validation complète + revue de code
Cadre + audit d'implémentation · 2–3 semaines
Tout ce qui est inclus dans la validation complète de l'audit de performance, plus une revue ligne par ligne de la logique du signal. Nous reconstruisons le signal indépendamment pour vérifier la reproductibilité, tester la contamination par anticipation et valider les hypothèses de coûts contre des données de marché réelles.
Toutes les couches de validation complète · Revue de code · Reconstruction du signal · Détection d'anticipation · Vérification des coûts

Mode B
Audit multi-stratégies
Book complet · 3–5 stratégies · 4–6 semaines
Validation complète et revue de code appliquées à l'ensemble d'un book de stratégies. Analyse de corrélation inter-stratégies, décomposition du risque au niveau portefeuille et recommandations de sizing. Produit un document de due diligence destiné aux investisseurs pour les fonds préparant une allocation ou une levée de capital.
Validation complète + code sur chaque stratégie · Corrélation inter-stratégies · Risque portefeuille · Recommandations de sizing · Document DD investisseur

Comment ça fonctionne

De la soumission au verdict

1
Soumission
Décrivez votre stratégie et envoyez votre historique de trades. Nous évaluons le périmètre et la faisabilité des données.
2
Proposition
Sous 48 heures, vous recevez une proposition à prix fixe avec le mode d'engagement recommandé et le calendrier.
3
Validation
Nous passons votre stratégie à travers le cadre propriétaire. Chaque couche produit un verdict binaire étayé par des preuves.
4
Rapport
Vous recevez un rapport structuré avec les verdicts, les diagnostics de risque et — pour les engagements complets — un débrief en direct.

Décrivez votre stratégie et le périmètre de validation souhaité. Nous renvoyons une proposition sous 48 heures.
